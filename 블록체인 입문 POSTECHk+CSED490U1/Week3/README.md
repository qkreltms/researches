## 3.1. Public & Private Blockchain

- Public Blockchain

  - Permissionless blockchain
  - 누구나 참여 가능
  - 누구나 transactions 기록 보기 가능
  - computing과 networking operations 에 참여하면 보상 받음
  - 비트코인, 이더리움, ...

- Private Blockchain
  - 관계자에게 승인 받아 참여
  - 승인 받은 nodes만 참여하고 거래 내역을 볼 수 있음
  - 보상이 없음
  - Hyperledger Fabric, R3, ...
